This is a fork of a fork of Crosspoint-Reader meant to implement Free 2 compatibility with the 1.1 firmware. I have included a firmware.bin file that can be downloaded and flashed using the usual website, but you should only do that if you're open to some degree of risk, as I have only tested this on my own device and no one has reviewed the code but me.

For the official Crosspoint-Reader firmware with a vetted release and clear instructions, please visit [the main repo](https://github.com/crosspoint-reader/crosspoint-reader)
